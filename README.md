# Supermart-Analysis

📌 Overview

End-to-end data analysis project to uncover insights on sales, customers, products, and returns.


🧰 Tools

Excel • Power Query • Python (Pandas) • PostgreSQL • Power BI


⚙️ Workflow

Data cleaning & transformation
Data ingestion into SQL
Exploratory Data Analysis (EDA)
Business analysis using SQL
Dashboard creation in Power BI


📊 Key Insights
📅 Seasonal sales spike (year-end)
👥 ~14% inactive customers
🏬 Store performance varies significantly
🔄 High sales ≠ low returns for some products


💡 Recommendations
Focus on high-performing products
Reduce return rates
Re-engage inactive customers
Optimize underperforming stores
